# Assignment 3 - Library Book Issue System

This folder contains a Java Swing implementation of the Library Book Issue System with custom exception handling.

## Files

- `src/LibraryManagementSystem.java`
- `src/EmptyFieldException.java`
- `src/InvalidRollNumberException.java`
- `src/InvalidDateException.java`
- `src/NullSelectionException.java`

## Features

- Java Swing GUI
- Labels, text fields, combo box, radio buttons, and text area
- `try-catch`, `throw`, and `finally`
- Custom exception classes
- `JOptionPane` success and error messages

## Run

From the `src` folder, compile and run with:

```bash
javac *.java
java LibraryManagementSystem
```
